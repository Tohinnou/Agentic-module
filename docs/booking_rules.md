# Règles de réservation

Une réservation peut être effectuée en ligne ou par téléphone.

Pour confirmer une réservation, le client doit fournir son nom, son numéro de téléphone, la date souhaitée, le créneau horaire et le type d’équipement demandé.

Une réservation peut être modifiée jusqu’à 24 heures avant le départ, sous réserve de disponibilité.

Les modifications demandées moins de 24 heures avant le départ ne sont pas garanties.

Un client ne peut pas réserver plus de trois équipements nautiques sur le même créneau sans validation manuelle de l’équipe.

Les clients doivent arriver 20 minutes avant l’heure de départ pour les formalités et les consignes de sécurité.
