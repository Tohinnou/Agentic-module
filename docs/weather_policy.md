# Politique météo

Marina Rentals peut annuler ou reporter une activité en cas de météo dangereuse.

Les situations concernées incluent les orages, les vents forts, une visibilité réduite ou une alerte officielle des autorités locales.

Si Marina Rentals annule l’activité pour raison météo, le client peut choisir entre un report gratuit ou un remboursement complet.

Si le client annule de lui-même alors que l’activité est maintenue, la politique d’annulation standard s’applique.

L’équipe support ne doit pas confirmer une activité en cas d’alerte météo sans validation du responsable opérationnel.
