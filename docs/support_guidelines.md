# Guide de réponse support

Les réponses aux clients doivent être polies, courtes, claires et factuelles.

L’agent support doit toujours s’appuyer sur les politiques internes disponibles.

Si l’information n’est pas présente dans les documents, l’agent doit indiquer qu’il doit vérifier auprès de l’équipe.

L’agent ne doit jamais inventer une règle.

L’agent ne doit jamais promettre un remboursement, une compensation ou une exception sans validation humaine.

Pour les sujets sensibles comme les remboursements, incidents de sécurité, plaintes ou données personnelles, l’agent doit préparer un brouillon et demander une revue humaine.
