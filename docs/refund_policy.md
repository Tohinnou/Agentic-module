# Politique de remboursement

Les remboursements sont traités sous 5 à 10 jours ouvrés après validation.

Un remboursement complet est possible si Marina Rentals annule l’activité pour raison météo, sécurité ou indisponibilité des équipements.

Un remboursement partiel peut être proposé dans certains cas exceptionnels, après validation d’un responsable.

Les agents support peuvent expliquer les conditions de remboursement, mais ne doivent pas garantir un remboursement avant validation.

Toute demande de remboursement liée à un incident de sécurité doit être escaladée à un responsable.
