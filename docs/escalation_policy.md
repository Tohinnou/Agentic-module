# Politique d’escalade

Une demande doit être escaladée à un responsable dans les cas suivants :

- demande de remboursement exceptionnel ;
- incident de sécurité ;
- plainte formelle ;
- client agressif ou menaçant ;
- demande liée aux données personnelles ;
- demande de réservation de groupe de plus de trois équipements ;
- contestation de frais de dommage matériel.

L’agent support doit indiquer clairement la raison de l’escalade dans le ticket.

L’agent ne doit pas clôturer un ticket escaladé sans retour du responsable.
